# Crypto Tracker

## 개발 목표
Styled-Components와 Typescript에 대해 배우고 React-Router-DOM과 React-Query, Recoil을 이용하여 리액트에 대한 더욱 심화적인 학습과 개발방법을 익히기 위해 개발.

## 사용 기술
- React.js
- Typescript
- React-Router-DOM v6
- React-Query
- Styled-Components
